# Password Generator (Python)

Jednoduchý a bezpečný generátor hesel napsaný v Pythonu. Projekt pro generování náhodných hesel s volitelnými parametry (délka, znaky apod.).

## Funkce

- Generování náhodných hesel s nastavitelnou délkou
- Volby zahrnutí: velká písmena, malá písmena, číslice, speciální znaky

## Rychlé spuštění

1. Naklonuj repozitář:
   ```
   git clone https://github.com/kofronadam/password-generator.git
   cd password-generator
   ```
2. Spusť CLI:
   ```
   python app.py
   ```
3. Postupuj podle pokynů pro generování hesla.
